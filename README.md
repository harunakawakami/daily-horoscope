## Installation

Install daily-horoscope with git clone or folk

## Run Locally

Clone the project

You can `git clone` and go to the project directory

```bash
  cd daily-horoscope
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
